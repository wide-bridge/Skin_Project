# 06_작업지시서_(최소챗봇_구현_및_VLM_RAG_연결준비)

## 1. 목적

1차 마무리 단계로 가장 간단한 챗봇을 먼저 구현하여, 이후 VLM과 RAG를 순차적으로 연결할 수 있는 서비스 루프를 만든다.

이번 06 단계의 목표는 아래와 같다.

- 최소 텍스트 챗봇 루프 구현
- FastAPI 또는 간단 UI 기준 대화 경로 확보
- 이후 VLM 결과와 RAG 검색 결과를 붙일 수 있는 응답 구조 확보

## 2. 선행 조건

- 04의 VLM 진단 결과 포맷 정의
- 05의 RAG corpus 및 retrieval 구조 정의

## 3. 구현 원칙

- 처음에는 가장 단순한 챗봇으로 시작
- 텍스트 입력/응답 루프 먼저 구현
- 이후 진단 결과가 들어오면 응답 payload에 결합
- 이후 RAG 검색 결과가 들어오면 설명 필드에 결합

## 4. 최소 기능

- 사용자 질문 입력
- 기본 응답 생성
- 세션 또는 최근 대화 유지
- 향후 `diagnosis_result`, `retrieved_contexts` 필드 확장 가능 구조

## 5. 완료 기준

- 최소 챗봇 요청/응답 루프 동작
- 후속 VLM 연결 포인트 확보
- 후속 RAG 연결 포인트 확보
- 1차 서비스 데모 가능한 최소 상호작용 확보
